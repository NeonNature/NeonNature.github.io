Mobile Money Website

A Project created for NCC Edu., (and yea, I would fail if I did not xD) using HTML, PHP and a little bit of CSS (Trust me, I didn't use that much CSS before and I regret that).

This is a web-version of Mobile Money applications with somewhat less complicated Logic. Super Agents will be holding the majority of cash and Agents of several shops/ cities will request some cash from Super Agents (1 per city or so), so that Agents can transfer those cash to Customers. Customer Types such as Golden Customers exist but discounts are yet to be implemented. Note that these accounts need valid Phone number (won't send SMS though) to be created. There might be several stuffs I forgot to mention.

Fortunately, this one was backed up before the HDD failure and I can still show it to those who wish along with documentations and Entity Relationship Diagrams. Not planning to host this using a free Dynamic web host unless requested.
